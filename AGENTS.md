# Hitanshi Chheda Portfolio — Agent Notes

This is a personal portfolio/resume website for Hitanshi Chheda.

The site uses a creative Excel-spreadsheet UI aesthetic to present professional experience, education, and skills.

## Project Structure

- `app/` — Next.js App Router pages and layouts
- `components/` — Reusable React components
- `public/` — Static assets (images, JSON data)
- `hero.ts` — Hero section data
- `makesomething.json` — Site configuration/content
