# Hitanshi Chheda — Personal Portfolio

A personal resume/portfolio website for Hitanshi Chheda, built with Next.js.

The site is designed as an interactive Excel-style spreadsheet showcasing her professional experience, education, and skills.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Tech Stack

- [Next.js](https://nextjs.org/) 16
- [HeroUI](https://heroui.com/) for UI components
- [Framer Motion](https://www.framer.com/motion/) for animations
- [Tailwind CSS](https://tailwindcss.com/) for styling
